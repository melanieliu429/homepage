My Work

Memory Card

I leveraged React hooks, including useState and useEffect, to manage state effectively and handle data fetching from an external API. This allowed me to create a dynamic user interface with shuffled images, a responsive scoreboard, and a best score tracker, showcasing my ability to build interactive and data-driven applications.

CV Application

I built a React application that allows users to input their information and generate a CV/résumé. I structured the app into components, effectively managing state and props to handle form inputs and enable editing and submission of the data.

Weather App

I built a weather forecast site using the Visual Crossing API, where users can search for a location and toggle between Fahrenheit and Celsius. The site dynamically updates the display based on the weather data, including changing backgrounds or adding weather-related images.

To Do List

I created a todo list application using object-oriented programming to dynamically manage todos with properties like title, description, and priority. The app supports multiple projects, editing todos, and data persistence using the Web Storage API, with a clear separation between application logic and DOM manipulation.

Restaurant Page

I practiced DOM manipulation by dynamically rendering a restaurant homepage using JavaScript to generate the entire website content. The styling was handled separately in a CSS file, allowing me to focus on building the structure with JavaScript.

Tic Tac Toe

Library
